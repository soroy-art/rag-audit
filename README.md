## Codebase

1. Clone the repository
2. Setup the Python virtual environment using Anaconda environment YAML files: qumls (other codes) or rag_chatbot2 (RAG models)
3. Main RAG model code is present in the root directory itself at: MAIN_CODE_rag_chroma_deepseek_qwen3_vectordb.ipynb
4. Baseline models code and data files can be found under "accountable_evidence_selection" directory
5. Evaluation datasets and results can be found under the respective directories: Qpain-risk-factors and Guideline-based Clinical QA benchmark
